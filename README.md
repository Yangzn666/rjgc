# 软件工程导论复习系统

## 项目概述
这是一个为软件工程导论课程设计的复习系统，帮助学生系统地复习各个章节的知识点，为期末考试做好充分准备。

## 功能特点
- 按章节分类展示软件工程导论的核心知识点
- 提供交互式学习体验，可点击展开详细内容
- 包含多种题型的练习测试
- 清晰的章节导航和页面跳转
- 重点内容用五角星（★）标记

## 目录结构
```
复习系统/
├── index.html          # 主页
├── chapter1.html       # 第一章：软件工程概述
├── chapter2.html       # 第二章：可行性研究
├── chapter3.html       # 第三章：需求分析
├── chapter5.html       # 第五章：总体设计
├── chapter6.html       # 第六章：详细设计
├── chapter7.html       # 第七章：实现（编码与测试）
├── chapter8.html       # 第八章：软件维护
├── chapter9.html       # 第九章：面向对象方法学
├── test.html           # 测试练习页面
├── keypoints.html      # 重点知识汇总
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── main.js         # JavaScript交互功能
└── images/             # 图片资源目录
```

## 使用方法
1. 直接在浏览器中打开 `index.html` 文件
2. 通过导航菜单访问不同的章节内容
3. 点击"查看详情"按钮展开知识点详细内容
4. 访问"测试练习"页面进行自我检测
5. 查看"重点知识汇总"页面复习核心知识点

## 技术特性
- 响应式设计，支持不同屏幕尺寸
- 交互式内容展开/收起功能
- 在线测试和答案验证功能
- 用户答题进度跟踪

## 浏览器兼容性
支持所有现代浏览器（Chrome, Firefox, Safari, Edge等）

## 开发说明
本系统使用纯HTML、CSS和JavaScript开发，无需服务器支持，可直接在本地运行。
